# Updated Vim setup
 
Run:
```
git clone git://github.com/cdwetzelberger/vimsetup2 ~/.vim
```

Create a .vimrc in your home directory with the line:

```
runtime vimrc
```

Install Vundle:
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

Install plugins
```
vim +PluginInstall +qall
```
